Learn how to make headings in HTML

Now that we have our basics down, let's start learning how to write content on our page.

The first tag we're going to learn after our basics is the heading tag. This is written as <h1>____</h1>. It is a double-sided tag that does not have any special attributes. It creates headings on a page. Here's an example:

<h1>This is a heading.</h1>
